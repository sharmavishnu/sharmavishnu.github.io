Detailed Description

The PixelFont interface. A font is created using a collection of pixels. Every font is an array of UInt32 numbers. So a 5x7 font, means, width is 5 pixels and height is 7 pixels. The array will contain, 7 UInt32 numbers. Each bit represents a pixel and reading left-2-right, if bit is 1, then pixel is switched on else pixel is switched off.

One can design their own font using the provided 'Font Designer Tool'. The final output from the tool is a *.cs file that contains the required definitions. It's a class that implements this interface.
