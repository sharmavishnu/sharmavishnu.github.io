Bmp280 is a pressure, temperature and altitude measuring sensor from Bosch. At the time of writing this code, the data was available at https://ae-bst.resource.bosch.com/media/_tech/media/datasheets/BST-BMP280-DS001.pdf

override void nanoFramework.Companion.Drivers.Sensors.Bmp280.Initialize ( )

Initialize the sensor. The current defaults, with which the sensor is inialized are:

  1. Power mode - Normal
  2. Pressure sampling - Standard
  3. Temperature sampling - Standard
  4. Standby - 125 ms
  5. Filter coefficient - X2
